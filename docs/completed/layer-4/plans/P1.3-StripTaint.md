## Test Descriptions

### Test 1: TestTaintEngine_StripTaint_Basic

**Given**: A map with taints `["PII", "SECRET", "TOOL_OUTPUT"]` and forbidden list `["PII", "SECRET"]`  
**When**: StripTaint is called with forbidden taints  
**Then**: Result has only `["TOOL_OUTPUT"]` taint  
**Expected Result**: Forbidden taints removed, allowed taints preserved  
**Spec Reference**: arch-v1.md L284 (inner → DMZ/outer auto-strip), L298 (stream sanitization)

### Test 2: TestTaintEngine_StripTaint_ReturnsStripped

**Given**: A map with taints `["PII", "SECRET", "TOOL_OUTPUT"]` and forbidden list `["PII"]`  
**When**: StripTaint is called with forbidden taints  
**Then**: Function returns stripped list `["PII"]` as second return value  
**Expected Result**: Stripped list matches forbidden taints that existed in object  
**Spec Reference**: arch-v1.md L284 (auto-strip), L286 (violation tracking)

### Test 3: TestTaintEngine_StripTaint_Nested

**Given**: A nested map structure with forbidden taints at multiple levels  
**When**: StripTaint is called with forbidden taints `["INNER_ONLY"]`  
**Then**: All nested levels have forbidden taints stripped; stripped list aggregated from all levels  
**Expected Result**: Forbidden taints stripped from all nested levels  
**Spec Reference**: arch-v1.md L283 (taint propagation), L284 (auto-strip)

### Test 4: TestTaintEngine_StripTaint_NoForbidden

**Given**: A map with taints `["TOOL_OUTPUT", "EXTERNAL"]` and forbidden list `["PII", "SECRET"]`  
**When**: StripTaint is called with forbidden taints not present in object  
**Then**: Object unchanged; empty stripped list returned  
**Expected Result**: No modification when no forbidden taints present  
**Spec Reference**: arch-v1.md L284 (conditional strip based on taints present)

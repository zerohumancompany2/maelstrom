## Test Descriptions

### Test 1: TestReportTaints_GeneratesTaintMap

**Given**: A runtime with tainted data: `{"user": ["PII"], "secret": ["SECRET"]}`  
**When**: ReportTaints("runtime-1") is called  
**Then**: Returns TaintMap `{"user": ["PII"], "secret": ["SECRET"]}`  
**Expected Result**: Complete TaintMap generated for all tainted objects  
**Spec Reference**: arch-v1.md L286 (taint propagation), L1303-1309 (ReportTaints interface)

### Test 2: TestReportTaints_RecursiveCollection

**Given**: A runtime with nested tainted data: `{"config": {"inner": {"key": ["SECRET", "INNER_ONLY"]}}}`  
**When**: ReportTaints("runtime-1") is called  
**Then**: Returns TaintMap with nested paths: `{"config.inner.key": ["SECRET", "INNER_ONLY"]}`  
**Expected Result**: Taints collected recursively from nested map structures  
**Spec Reference**: arch-v1.md L283 (nested taint propagation), L286 (taint tracking)

### Test 3: TestReportTaints_EmptyForUntainted

**Given**: A runtime with no tainted data (empty or clean objects)  
**When**: ReportTaints("runtime-1") is called  
**Then**: Returns empty TaintMap `{}`  
**Expected Result**: Empty TaintMap returned for untainted runtime  
**Spec Reference**: arch-v1.md L286 (taint tracking), L1303-1309 (ReportTaints interface)

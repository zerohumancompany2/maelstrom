## Acceptance Criteria

1. Tool with `taintOutput: true` or `taintOutput: ["TOOL_OUTPUT"]` auto-attaches taints on tool_result
2. Tool result inherits tool's boundary for taint context
3. Tool with `taintOutput: false` or empty list does not auto-attach taints

## Test Descriptions

### Test 1: TestToolTaintOutput_AutoAttach

**Given**: A tool configured with `taintOutput: ["TOOL_OUTPUT"]` and boundary `dmz`  
**When**: Tool executes and returns a result  
**Then**: The tool_result Mail has `Metadata.Taints` containing `["TOOL_OUTPUT"]`  
**Expected Result**: Tool output automatically receives configured taints  
**Spec Reference**: arch-v1.md L562 (taintOutput configuration), L1070 (automatic attachment)

### Test 2: TestToolTaintOutput_InheritBoundary

**Given**: A tool configured with `taintOutput: ["TOOL_OUTPUT", "INNER_ONLY"]` and boundary `inner`  
**When**: Tool executes and returns a result  
**Then**: The tool_result Mail has `Metadata.Boundary` set to `inner` and `Metadata.Taints` contains `["TOOL_OUTPUT", "INNER_ONLY"]`  
**Expected Result**: Tool result inherits tool boundary and receives all configured taints  
**Spec Reference**: arch-v1.md L566 (multiple taints), L564 (boundary configuration)

### Test 3: TestToolTaintOutput_NoAttach

**Given**: A tool configured with `taintOutput: false` or empty `taintOutput: []`  
**When**: Tool executes and returns a result  
**Then**: The tool_result Mail does NOT have automatic taints attached (empty or only manually added taints)  
**Expected Result**: Tool without taintOutput configuration does not auto-attach taints  
**Spec Reference**: arch-v1.md L562, L566 (taintOutput is optional configuration)

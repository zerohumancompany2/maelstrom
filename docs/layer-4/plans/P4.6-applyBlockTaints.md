### Test 1: TestSecurityService_applyBlockTaints_MergesSourceTaints

**Given**: Multiple ContextBlocks with different taint sets:
  - Block A: taints=["USER_SUPPLIED"]
  - Block B: taints=["TOOL_OUTPUT"]
  - Block C: taints=["MEMORY_INJECTED"]

**When**: applyBlockTaints is called with all blocks for boundary=dmz

**Then**: Resulting ContextBlock contains merged taints=["USER_SUPPLIED", "TOOL_OUTPUT", "MEMORY_INJECTED"]

**Expected Result**: Taints from all source blocks are unioned into single taint set

**Spec Reference**: arch-v1.md L279 (ContextBlock assembly), L283 (taint propagation)

### Test 2: TestSecurityService_applyBlockTaints_EnforcesTaintPolicy

**Given**: ContextBlocks with mixed taints for boundary=outer:
  - Block A: taints=["USER_SUPPLIED", "INNER_ONLY"]
  - Block B: taints=["TOOL_OUTPUT"]

**When**: applyBlockTaints is called with blocks for boundary=outer (INNER_ONLY forbidden)

**Then**: Block A is dropped/redacted (contains forbidden INNER_ONLY), Block B passes through

**Expected Result**: Blocks with forbidden taints for target boundary are filtered out

**Spec Reference**: arch-v1.md L285 (Inner → DMZ/outer: auto-strip or block), L293 (forbidden taint redaction)

## Acceptance Criteria

1. Orchestrator spawns isolated execution context for each tool call
2. Isolated context uses separate namespace (not parent ChartRuntime)
3. Tool execution in isolated context cannot access parent state directly
4. Results from isolated context injected as tainted Messages to parent session

**Given**: A parent ChartRuntime with mutable state including sensitive data  
**When**: Tool executes in isolated context and attempts to read parent state

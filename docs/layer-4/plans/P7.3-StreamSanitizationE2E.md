## Test Descriptions

### Test 1: TestE2E_StreamSanitization_MultiChunkLatency

**Given**: 
- DMZ agent streaming response to outer client via SSE (arch-v1.md L664-666)
- Stream configured with `stream: true` in expose block (L717)
- 10 chunks of varying sizes (1KB to 10KB each)
- Each chunk has taints attached (mix of allowed and forbidden)

**When**: Agent streams response → each chunk passes through Gateway sanitization  
**Then**:
- Each chunk processed independently (stateless, no full buffering) (L300)
- Per-chunk sanitization latency measured <50ms (L299-300)
- All 10 chunks delivered with sequence numbers preserved (L645)
- `isFinal` flag correctly set on last chunk (L646)
- Total stream latency = sum of per-chunk latencies (not buffered)
**Expected Result**: Multi-chunk stream sanitized with <50ms per-chunk latency end-to-end  
**Spec Reference**: arch-v1.md L298-301 (stream sanitization), L644-648 (StreamChunk format), L1518 (SHALL strip forbidden taints)

### Test 2: TestE2E_StreamSanitization_PII_Redaction

**Given**: 
- DMZ agent streaming assistant response containing PII data
- Stream chunks have taints: `["TOOL_OUTPUT", "PII", "INNER_ONLY"]`
- taintPolicy with `allowedOnExit: ["TOOL_OUTPUT"]` (PII and INNER_ONLY forbidden) (L210-213)
- redactRules configured for PII and INNER_ONLY replacement (L214-216)

**When**: Stream sent from DMZ agent to outer client via Gateway  
**Then**:
- Each chunk passes through DMZ sanitizers (L298)
- Chunks with PII taint → content redacted per redactRules (e.g., `[PII REDACTED]`)
- Chunks with INNER_ONLY taint → content stripped or redacted
- Chunks with only TOOL_OUTPUT taint → passed through unchanged
- Outer client receives stream with forbidden taints removed/redacted
- Stream metadata.taints field updated to reflect remaining allowed taints only
- No PII or INNER_ONLY content visible in delivered chunks
**Expected Result**: Stream sanitization strips forbidden taints per-chunk with redaction  
**Spec Reference**: arch-v1.md L298 (DMZ sanitizers), L214-216 (redactRules), L1518 (SHALL strip forbidden taints from stream chunks)

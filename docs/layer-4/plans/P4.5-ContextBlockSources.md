### Test 1: TestContextBlockSource_StaticContent

**Given**: A ContextBlock with `source: static` and content field set to "You are a secure agent"  
**When**: AssembleSource is called on the block  
**Then**: Block returns the static content exactly as configured  
**Expected Result**: Static content is returned without modification  
**Spec Reference**: arch-v1.md L995-999 (static source), L2028 (ContextBlock entity)

### Test 2: TestContextBlockSource_SessionLastN

**Given**: A ContextBlock with `source: session`, `strategy: lastN`, `n: 30` and session with 50 messages  
**When**: AssembleSource is called on the block  
**Then**: Block returns last 30 messages from session, in chronological order  
**Expected Result**: Session source correctly retrieves last N messages  
**Spec Reference**: arch-v1.md L1001-1005 (session source with lastN strategy), L2028 (ContextBlock entity)

### Test 3: TestContextBlockSource_MemoryServiceRAG

**Given**: A ContextBlock with `source: memoryService`, `query: "{{currentGoal}}"`, `topK: 8`  
**When**: AssembleSource is called with currentGoal = "implement feature X"  
**Then**: Block returns top 8 memory results matching the query  
**Expected Result**: Memory service query returns relevant memories within topK limit  
**Spec Reference**: arch-v1.md L1015-1020 (memoryService source), L2028 (ContextBlock entity)

### Test 4: TestContextBlockSource_ToolRegistryBoundaryFilter

**Given**: A ContextBlock with `source: toolRegistry`, `boundaryFilter: dmz`  
**When**: AssembleSource is called at DMZ boundary  
**Then**: Block returns only tools marked as visible at DMZ boundary  
**Expected Result**: Tool registry filters by boundary correctly  
**Spec Reference**: arch-v1.md L1023-1027 (toolRegistry source with boundaryFilter), L2028 (ContextBlock entity)

### Test 5: TestContextBlockSource_UnknownSourceType

**Given**: A ContextBlock with `source: unknownType` (not static/session/memoryService/toolRegistry/runtime)  
**When**: AssembleSource is called on the block  
**Then**: Error is returned indicating unsupported source type  
**Expected Result**: Appropriate error for unknown source types  
**Spec Reference**: arch-v1.md L995-1030 (defined source types), L2028 (ContextBlock entity)

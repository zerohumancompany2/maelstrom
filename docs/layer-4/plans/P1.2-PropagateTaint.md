## Test Descriptions

### Test 1: TestTaintEngine_PropagateTaint_MapMerge

**Given**: A map with existing `_taints: ["PII"]`  
**When**: PropagateTaint is called with new taints `["TOOL_OUTPUT"]`  
**Then**: Result has `_taints: ["PII", "TOOL_OUTPUT"]` with no duplicates  
**Expected Result**: Existing taints merged with new taints  
**Spec Reference**: arch-v1.md L283 (taint propagation), L1305 (TaintPropagate interface)

### Test 2: TestTaintEngine_PropagateTaint_NestedMap

**Given**: A nested map structure with taints at root level  
**When**: PropagateTaint is called with new taints `["SECRET"]`  
**Then**: All nested map levels receive the merged taint set  
**Expected Result**: Taints propagated to all nested map levels, each level has complete taint set  
**Spec Reference**: arch-v1.md L283 (propagation on copy/read/write)

### Test 3: TestTaintEngine_PropagateTaint_Slice

**Given**: A slice containing map elements with existing taints  
**When**: PropagateTaint is called with new taints `["WORKSPACE"]`  
**Then**: All map elements in slice receive merged taints; non-map elements unchanged  
**Expected Result**: Taints propagated to all map elements in slice  
**Spec Reference**: arch-v1.md L283 (taint propagation), implementation-assessment.md L280-285

### Test 4: TestTaintEngine_PropagateTaint_Deduplication

**Given**: An object with existing taints `["PII", "SECRET"]`  
**When**: PropagateTaint is called with overlapping taints `["SECRET", "TOOL_OUTPUT"]`  
**Then**: Result has `["PII", "SECRET", "TOOL_OUTPUT"]` with SECRET appearing only once  
**Expected Result**: Duplicate taints removed, taint order preserved for unique values  
**Spec Reference**: arch-v1.md L273 (taint sets), L283 (propagation)

### Test 5: TestTaintEngine_PropagateTaint_EmptySource

**Given**: An object without existing taints or empty newTaints list  
**When**: PropagateTaint is called with new taints or empty list  
**Then**: Object without taints receives new taints; empty newTaints preserves existing  
**Expected Result**: Correct handling of edge cases with empty sources  
**Spec Reference**: arch-v1.md L276 (security invariant), L283 (propagation)

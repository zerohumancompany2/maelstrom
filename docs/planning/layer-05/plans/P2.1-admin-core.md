**Given**: An AdminService instance with multiple running agents in the runtime registry

**Expected Result**: Agent state included in response, all agents enumerated correctly  
**Spec Reference**: arch-v1.md L467 (k9s-style terminal/web console for debugging), L485 (adminCommand: list)

**Given**: An AdminService instance and a running agent with known state  
**When**: ControlAgent() is called with pause/resume/stop action  
**Then**: Agent state transitions correctly  
**Expected Result**: Pause/resume/stop commands executed, agent state updated in registry  
**Spec Reference**: arch-v1.md L467 (debugging console), L485 (adminCommand: control)

### Test 4: TestAdminService_QueryTaints

**Given**: An AdminService instance and an agent with taints in its ContextMap  
**When**: QueryTaints() is called for the specified agent  
**Then**: TaintMap returned with all taints for that agent  
**Expected Result**: All taints enumerated, taint sources and values included  
**Spec Reference**: arch-v1.md L467 (debugging console), L485 (adminCommand: queryTaints)
